# Auto-Return Classification Board

A responsive classification board application built with React, TypeScript, and Vite. The application allows users to organize, classify, and manage items across multiple categories through an intuitive board interface.

## Features

* Display items grouped by category
* Move items between categories
* Assign and unassign classifications
* Responsive layout for desktop and mobile devices
* Countdown indicators for item expiration (if enabled)
* Type-safe implementation with TypeScript
* Clean and maintainable architecture

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Vitest
* ESLint

## Getting Started

### Prerequisites

* Node.js >= 20
* Yarn

### Installation

```bash
yarn install
```

### Run Development Server

```bash
yarn dev
```

### Build for Production

```bash
yarn build
```

### Preview Production Build

```bash
yarn preview
```

### Run Tests

```bash
yarn test
```

### Run Linting

```bash
yarn lint
```
